                        Wer hat einen erh\u00f6hten Tagesbedarf an Vitamin C?<\/h3>

                          • Schwangeren<\/span><\/b> Frauen wird ab dem vierten Schwangerschaftsmonat eine t\u00e4gliche Zufuhr von 105 mg empfohlen, w\u00e4hrend der Tagesbedarf in der Stillzeit sogar auf 125 mg ansteigt<\/span>.<\/span><\/span>2<\/sup><\/span><\/span><\/span><\/li><\/ul>\t\t\t\t\t\t\t\t<\/div>\n\t\t\t\t

                                            Welche Funktion hat Vitamin C im K\u00f6rper?<\/h2>

                                            Vitamin C hat im K\u00f6rper viele Funktionen. Es ist unter anderem unverzichtbar f\u00fcr die <\/span>Bildung von Kollagen<\/span><\/b>.                                               • einer normalen Funktion des Nervensystems<\/span>\u00a0<\/span><\/li><\/ul>
