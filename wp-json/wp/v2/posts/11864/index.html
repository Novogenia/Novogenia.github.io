Kronen Zeitung – Date: 2020\/07\/13Author: unknown – Source<\/a><\/p>\n\n\n\n

\n

The drugstore dm has announced that it will now offer coronavirus tests for home use. For security reasons, however, the tests are to be sold exclusively via the online store. These are so-called PCR tests, as previously announced by BIPA for the end of July. For 119 euros, the dm test can be carried out at home – and only via a saliva sample. The examination of the samples should then take place in certified laboratories.<\/p>\n<\/blockquote>\n\n\n\n
